Set Tedariği Nedir?

Lider Telsiz, film ve dizi setlerinin dinamik yapısını en iyi bilen ekiplerden biridir. Sadece telsiz vermekle kalmıyor, çekim alanındaki tüm haberleşme lojistiğini yönetiyoruz. Set Tedariği hizmetimiz, yönetmen grubundan prodüksiyona, teknik ekipten sanat grubuna kadar herkesin senkronize çalışmasını sağlar.

Özel batarya şarj istasyonlarımız, yedek cihaz paketlerimiz ve set disiplinine uygun kulaklık seçeneklerimizle, en zorlu gece çekimlerinde veya geniş arazi operasyonlarında iletişiminiz asla kesilmez.
